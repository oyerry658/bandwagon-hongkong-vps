## 延迟实际表现如何

直接说数字：

- 广东方向：约 9ms
- 华东（上海/杭州一带）：平均 39ms 左右
- 联通方向：零丢包，表现最稳
- 移动方向：部分节点有超时，广东广州、北京区域有时会碰到，主要用移动 SIM 卡的用户要注意
